# Social Media Post Design

A modern and visually appealing social media recruitment post created for a Graphic Designing Internship task.

## Project Overview
This project focuses on creating an engaging social media post using design principles such as:
- Typography
- Visual hierarchy
- Branding consistency
- Color harmony
- Composition

The design was created for a fictional **Graphic Design Intern hiring post** with a clean and aesthetic layout.

## Tools Used
- Canva

## Features
- Modern recruitment post design
- Soft transparent overlay for better readability
- Professional typography styling
- Consistent warm color palette
- Personal branding included
- Social media friendly square format

## Design Elements
- Bold heading for visual impact
- Clean information structure
- Warm aesthetic background image
- Balanced spacing and alignment
- Readable text hierarchy

## Skills Demonstrated
- Social Media Post Design
- Typography
- Layout Composition
- Color Theory
- Branding
- Visual Storytelling

## Project Outcome
This task helped improve:
- Creative thinking
- Design customization
- Readability optimization
- Professional social media design skills
